# Perjadin
Aplikasi Perjalanan Dinas

Note : Maaf source code tidak dapat diperlihatkan karena privasi (sudah dipakai oleh kemendikbud)

Perjadin merupakan aplikasi yang digunakan sebagai laporan perjalanan yang biasanya dilakukan oleh para pegawai dinas. Kadang mereka
membuat laporan begitu sangat repot dan juga acak-acakan dan juga kertas sebagai bahannya. Dengan aplikasi perjadin ini saya berharap
dapat membantu para pegawai dalam mengelola perjalanan mereka dengan cara melaporkan bukti-bukti perjalanan seperti tiket kereta, 
tiket pesawat, bon hotel, dll.

# Fitur
1. Login
2. Logout
3. Lihat Laporan
4. Laporan Baru
5. Dashboard Screen
6. Kamera untuk foto bukti-bukti
7. Lihat kehadiran
8. Lihat laporan
9. Splash screen
10. Profile User
11. Ambil Foto Kegiatan
12. Ambil Foto Bukti

# Teknologi
1. Java
2. MySql
3. PHP
